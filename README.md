# Applied Archive Atelier

A [Vite](https://vite.dev) + React + TypeScript single-page app.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

- `npm run dev` — start the dev server
- `npm run build` — type-check and build for production (output in `dist/`)
- `npm run preview` — preview the production build locally

## Project structure

- `src/pages` — one component per route (wired up in `src/App.tsx`)
- `src/components` — shared UI
- `src/lib` — project and team data
- `src/assets` — imported images
- `public` — static files served at the root (fonts, favicon)
